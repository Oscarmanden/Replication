How to run the system:

Start server1 and server2
By typing 
go run Server/Server1.go
go run Server/Server2/Server2.go

Start client
By typing go run Client/Node.go

Then to start the auction type start in server1 cli
to bid type an amount in the node cli
to get result type result in node cli

to simulate crashing the primary server type shutdown or pres ctrl+c
